# 💐 Сайт-приглашение на свадьбу

Элегантный веб-сайт для приглашения гостей на свадьбу с персонализацией, формой RSVP и интеграцией с Telegram.

---

## 🎉 Особенности

- ✨ **Персонализированные приглашения** - `?name=Имя` для именных ссылок
- ⏱️ **Обратный отсчет** до даты свадьбы в реальном времени
- 📝 **Форма RSVP** с подтверждением присутствия
- 📱 **Telegram интеграция** - ответы приходят в бот
- 🎨 **Элегантный дизайн** - цветочная тема с пастельными тонами
- 📱 **Адаптивный** - отлично выглядит на всех устройствах
- 🔒 **TypeScript** - полная типизация кода

## 🚀 Быстрый старт

### Локальная разработка


## ✏️ Что нужно изменить перед деплоем

### 1. Основная информация

В `src/components/MainInvitation.tsx`:

```typescript
// Дата свадьбы (строка 22)
const weddingDate = new Date('2026-07-31T16:00:00')

// Место проведения (строка 134)
<p>Ресторан "Название"<br />ул. Адрес, 123<br />Москва</p>

// Тайминг (строка 123-128)
// Дресс-код (строка 140)
```

### 2. Telegram настройки

Создайте `.env`:

```env
VITE_TELEGRAM_BOT_TOKEN=ваш_токен
VITE_TELEGRAM_CHAT_ID=ваш_chat_id
```

Или используйте безопасный backend (см. `TELEGRAM_BACKEND_SETUP.md`)

### 3. Фотографии

Замените `public/images/couple-1.jpg` на вашу фотографию

### 4. GitHub Pages

В `package.json`:

```json
"homepage": "https://your-username.github.io/your-repo",
```

## 📱 Форма RSVP

Форма собирает:
- ✅ Имя и Фамилия
- ✅ Будет ли присутствовать
- ✅ С супругой/супругом или нет

Отправляет в Telegram:
```
🎊 Новый ответ на приглашение

👤 ФИО: Иван Иванов
✅ Присутствие: Да, будет
👫 Сопровождение: С супругой/супругом
```
